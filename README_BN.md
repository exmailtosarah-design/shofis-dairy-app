- "Create repository" ক্লিক করুন

**ধাপ ২:** কোড আপলোড করুন
```bash
cd ~/shofis_dairy_app
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/shofis-dairy-app.git
git branch -M main
git push -u origin main
```

**ধাপ ৩:** APK ডাউনলোড করুন
- Repository এ যান
- "Actions" ট্যাবে ক্লিক করুন
- "Build Android APK" workflow খুলুন
- "Artifacts" থেকে `shofis-dairy-apk` ডাউনলোড করুন

**ধাপ ৪:** ফোনে ইনস্টল করুন
- APK ফাইল ফোনে কপি করুন
